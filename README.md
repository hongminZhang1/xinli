#  心晴驿站 - 青少年心理健康AI陪伴平台

##  项目简介
「心晴驿站」是一个专为青少年打造的心理健康AI陪伴与综合干预服务平台。面对当代青年群体多发的情绪压力、学业焦虑等心理困境，项目创新性地引入了 **DeepSeek 大模型**，构建了724小时在线、具备深度共情能力的专属AI心理伙伴「小晴」。

本项目旨在打破传统心理干预服务的时空与成本壁垒，将前沿AI技术与专业心理学相融合。平台集成了 **AI情感陪伴、专业心理测评、情绪追踪分析、沉浸式互动解压、心事日记** 等核心功能，致力于为每一个年轻的心灵提供安全、即时、专业的数字心理港湾，让青少年随时随地都能找到属于自己的晴朗天空。

---

##  核心功能亮点 (AI驱动)

###  1. 全天候AI心理伙伴「小晴」
* **实时倾听与疏导**：由 DeepSeek 强力驱动，支持自然语言对话，能够精准识别用户情绪，提供基于认知行为疗法（CBT）的温和回应。
* **秒级响应、无惧时差**：无论深夜还是清晨，用户都能获得专业的共情陪伴，成为最安全的私密树洞。

###  2. 智能心理测评系统
* 内置标准化抑郁、焦虑、压力等相关心理学测评量表。
* 结合 AI 数据分析能力，根据用户测试结果自动生成通俗易懂的个性化心理健康分析报告。

###  3. 情绪追踪与AI洞察
* **心情雷达**：支持每日打卡与情绪日志记录，生成个人成长趋势图表（Growth Trend）。
* **AI数据分析洞察**：系统通过长线记录自动捕捉用户情绪的起伏暗流，并由 AI 给予适当的反馈与干预提示。

###  4. 沉浸式互动解压空间
提供丰富的数字化解压工具箱与防脱焦辅助：
* **4-7-8 呼吸训练 (Breathing Exercise)**：科学调息，辅助身心放松。
* **禅意打字 (Zen Typing) & 色彩涟漪 (Color Ripples)**：通过沉浸式感官交互释放压力。
* **益智解压小游戏**：涵盖记忆卡牌 (Memory Cards)、滑块拼图 (Sliding Puzzle) 等，转移负面注意力。

###  5. 心理科普与互助社区
* **科普文章聚合**：汇聚专业心理健康文章资源，增加用户的心理学认知。
* **互助评论与圈子**：支持用户点赞、评论与情感分享，营造具有温暖正反馈的青年同辈支持网络。

###  6. 真人心理辅导衔接
* 当AI识别到高风险情绪或用户主动需求时，支持线上一键获取专业资源与心理干预建议。
* 衔接咨询师留言机制 (Counselor Inbox/Chat)，构建从 AI 初步干预到人工介入的完整闭环。

##  技术架构

本项目采用了现代化的全栈技术生态，确保极致的交互体验与系统响应：

* **前端框架**：Next.js 14 (App Router) + React
* **UI 与样式**：Tailwind CSS + Lucide Icons + Radix UI
* **后端支撑**：Next.js API Routes + Node.js
* **大模型引擎**：DeepSeek LLM (提供核心 AI 对话与分析)
* **数据库层**：Prisma ORM (兼容 MySQL / PostgreSQL)
* **用户认证**：NextAuth.js (提供安全统一的鉴权体系)
* **静态资源管理**：Uploadthing

##  快速启动

### 准备环境
1. 请确保已安装 **Node.js** (v18+) 
2. 请确保已安装包管理工具 **pnpm**

### 1. 安装依赖
\\\ash
pnpm install
\\\

### 2. 环境变量配置
复制项目根目录下的 \.env.example\ 为 \.env\ (如果存在)，并在其中填入必要的数据库连接字符串(DATABASE_URL) 和 AI 平台的 API 密钥(DEEPSEEK_API_KEY)。

### 3. 数据表同步 (Prisma)
\\\ash
npx prisma db push
\\\

### 4. 启动开发服务器
\\\ash
pnpm dev
\\\
启动成功后，在浏览器访问 [http://localhost:3000](http://localhost:3000) 即可预览。
